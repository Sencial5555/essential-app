## Definition
A knowledge organization system used inside [[Obsidian]] to structure project files so [[Claude Code]] can navigate them without guessing. PARA = Projects, Areas, Resources, Archives.

## The Four Categories

| Folder | Contents for Essential |
|--------|----------------------|
| **Projects** | Active code files: `index.html`, `server.js`, active features |
| **Areas** | Responsibilities: API account management, deployment, billing |
| **Resources** | Reference material: Sightengine API docs, Tailwind cheat sheets, CLAUDE.md |
| **Archives** | Completed or deprecated work |

## Why It Matters for AI Development
PARA makes Claude's instructions precise: "Look at the API documentation in my Resources folder, and use it to update the server code in my Projects folder." Claude does the job in seconds because it knows exactly where to look.

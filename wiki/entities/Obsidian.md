## Overview
Obsidian is a free, local markdown note-taking app that functions as the project control center and external memory for Claude. All notes are stored as plain `.md` files in a folder called a "Vault" — readable directly by [[Claude Code]] without copy-pasting.

## Key Facts
- Free to download; stores everything locally on disk
- Killer feature: CLAUDE.md at vault root = Claude's permanent memory across sessions
- Graph View: visual map of all linked notes — see which files connect to which
- Obsidian Web Clipper: browser extension to clip web articles/API docs directly to raw/ folder
- Prompt caching: Claude "freezes" the vault structure, reducing token usage by up to 70%

## Usage in Essential
- Left half of the [[Dual-Screen Workspace]]
- Holds CLAUDE.md, API documentation notes, feature plans, and this wiki
- [[PARA Framework]] organizes the vault into Projects, Areas, Resources, Archives
- Use Web Clipper to ingest Sightengine/Hive API documentation into raw/

## Overview
Cursor is an AI-first code editor — a heavily upgraded fork of VS Code — built specifically for pairing with AI. It serves as the visual workbench for the Essential project, sitting in the right half of the [[Dual-Screen Workspace]] while [[Obsidian]] occupies the left.

## Key Facts
- Role: visual interface for all project files; used for Phase 1 (frontend design)
- Killer feature: **Composer Mode** (Ctrl+I / Cmd+I) — type plain English, see code changes flash in front of you
- [[Claude Code]] runs inside Cursor's built-in bottom terminal panel
- Free to download; locally installed

## Usage in Essential
- Phase 1: Cursor Composer generates HTML + Tailwind CSS dark-themed mobile UI
- Phase 2+: bottom terminal runs Claude Code for autonomous backend/database work
- Top half of Cursor shows files being written automatically while Claude Code runs below

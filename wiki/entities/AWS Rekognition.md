## Overview
AWS Rekognition is Amazon's enterprise image analysis service. In the Essential long-term vision, it represents the infrastructure upgrade path once the app outgrows third-party APIs like Sightengine or Hive Moderation.

## Key Facts
- Role: ultimate infrastructure upgrade at massive scale; offers lowest raw computing costs globally
- Transition: hire a developer for a week to switch from Sightengine/Hive API to direct AWS/GCP integration
- Not relevant for MVP or early growth stages

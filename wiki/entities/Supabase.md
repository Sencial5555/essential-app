## Overview
Supabase is a cloud-based database platform (free tier available) recommended as the database for Essential in Phase 3. It lives entirely online — no local installation required.

## Key Facts
- Role in Essential: stores user profiles, email addresses, and scan credit counts
- Free tier available; Claude Code writes the connection code
- Alternative to Firebase and SQLite for the project database
- Access: sign up on website; Claude Code writes a few lines to connect the app to it over the internet

## Summary
Concise 3-step guide for building Essential: visual prototype with Claude Artifacts → backend + API connection → Vercel deployment. Written for someone with zero coding skills.

## Key Points
- Step 1: Use Claude Artifacts to build single-file HTML app with Tailwind CSS; see it live in chat.
- Step 2: Sign up for Sightengine free developer account → get API key → give Claude the backend prompt.
- Step 3: Ask Claude to walk through Vercel/GitHub deployment step by step.
- Entire initial MVP can be built without leaving the Claude chat interface.

## Summary
Comprehensive 7-step build workflow from environment setup through live deployment. The most complete single-document view of the entire Essential build process.

## Key Points
- Step 1: Environment — Obsidian vault + Claude Code terminal in project folder.
- Step 2: Karpathy wiki ingestion — generate raw/ and wiki/ folders, CLAUDE.md, ingest API docs.
- Step 3: Frontend — Cursor Composer, single-page HTML + Tailwind CSS, dark-themed mobile UI.
- Step 4: Backend — Claude Code writes Node.js/Python server, securely bridges to detection API using hidden keys.
- Step 5: Monetization — Stripe or Lemon Squeezy merchant account, paywall overlay, webhook script for credit unlocks.
- Step 6: Testing & guardrails — rate limits, unauthenticated bot blocks, payment simulation in test mode.
- Step 7: Deploy — Git init → GitHub push → Vercel/Render link → live URL in under 10 seconds.

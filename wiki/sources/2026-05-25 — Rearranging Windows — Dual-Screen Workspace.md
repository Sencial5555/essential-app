## Summary
Recommended screen layout for building Essential: Obsidian on the left, Cursor on the right, Claude Code running inside Cursor's bottom terminal. Phase-specific window setups and the "emergency room" use of browser Claude.

## Key Points
- Default layout: Obsidian (left half) for blueprint/notes, Cursor (right half) for files/code.
- Phase 1 (design): Cursor Composer panel open; Chrome tab in background to preview index.html.
- Phase 2–3 (backend/database): Cursor terminal at bottom running Claude Code; top half of Cursor shows files being autonomously written.
- Browser Claude tab: emergency fallback only — paste broken code and say "look at this with fresh eyes."
- The layout means you never switch apps to check rules; Obsidian is always visible on the left.

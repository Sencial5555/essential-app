## Summary
Comprehensive guide to using Obsidian as the project control center: what it is, how it pairs with Claude Code, the PARA framework for organization, and the full tool workflow loop from Obsidian to live Vercel deployment.

## Key Points
- Obsidian: free, local markdown files. Stores everything as plain text — AI reads it directly.
- Killer feature: CLAUDE.md at vault root = permanent AI memory. Both Cursor and Claude Code read it automatically.
- Cursor: AI-first code editor (VS Code fork). Composer Mode (Ctrl+I / Cmd+I) for plain English design changes.
- Claude Code: autonomous developer inside Cursor terminal; `/goal` command for high-level tasks.
- Full workflow loop: Obsidian (plan) → Cursor (view) → Claude Code (build) → GitHub (save) → Vercel (deploy live).
- Lemon Squeezy or Stripe: sign up, generate payment link, Claude Code writes webhook to update database credits on purchase.

## Summary
Long-term financial model for Essential at scale. Covers API overage behavior, enterprise volume discount tiers, the 1-million-scan profit math, and the ultimate upgrade path to raw cloud infrastructure.

## Key Points
- No hard caps on paid API plans — Sightengine uses overage rates (~$0.0015–$0.0020/scan) rather than shutting the app down.
- Enterprise Tier: at millions of scans, negotiate custom contract with API provider's sales team; volume discounts can cut cost per scan by 50%+.
- 1M scan month math: 10,000 users × $4.99 = $49,900 revenue; minus $5,500 payment fees + $1,000 API costs = ~$43,400 net profit.
- Payments scale with costs: users pay before API bills arrive, so cash flow is always positive.
- Ultimate upgrade: at sufficient scale, transition from Sightengine/Hive to AWS Rekognition or Google Cloud AI for lowest raw compute costs.

## Summary
Catalog of the full Claude tool ecosystem and how each component helps build Essential. Covers development tools, organization features, automation, and intelligence controls.

## Key Points
- Claude Code: terminal engineer; `/goal` command for autonomous multi-step tasks; can spawn background sub-agent teams.
- Claude in Chrome: browser extension; reads live console errors, network requests, DOM — fixes visual/JS bugs by looking at the live site.
- Projects & Cowork: persistent workspaces with pinned files; sub-chats inherit same knowledge base; prevents context loss.
- Artifacts: live interactive sandbox beside chat; clickable previews before deploying; up to 20MB persistent storage.
- MCP (Model Context Protocol): universal connector to external tools (Perplexity for web search, Google Drive, filesystems).
- Custom Skills & Routines: slash commands + scheduled background automations on Anthropic cloud (run even when laptop is off).
- Adaptive Reasoning: Claude dynamically scales compute — simple tasks skip thinking, complex security tasks use max effort.
- Context Compaction: auto-summarizes older conversation parts to prevent freezes on long sessions.

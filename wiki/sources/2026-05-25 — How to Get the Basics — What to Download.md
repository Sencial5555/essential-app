## Summary
Clarifies the minimal local installation footprint needed for the project. Only 4 things need to be downloaded to the physical computer; everything else (Tailwind, database, hosting) lives in the cloud.

## Key Points
- Download list: Google Chrome, Obsidian, Cursor, Node.js — that's it.
- HTML and CSS are languages built into every browser, not programs to install.
- Tailwind: loaded via CDN link in the HTML file — no local install.
- Database (Supabase or Firebase): lives entirely in the cloud; Claude Code writes the connection code.
- Node.js is the only technical dependency that must be manually installed locally.
